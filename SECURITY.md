# Security Policy

## Reporting a Vulnerability

This repository only contains plain text files. Executables or anything which invokes an executable should not be contained within it.

There is an NTA2 application for Windows. If you setup the default application association on your Windows PC, to launch it? Opening the `.tri` file is (presumed) safe and reliable.

If you discover a security issue, please report it by email to glen.ritchie.usa@gmail.com address. We will respond within 4 business days.

## Disclosure Policy

Please do not publicly disclose security vulnerabilities until we have had a chance to address them. We appreciate responsible disclosures and will keep you updated on our progress.
